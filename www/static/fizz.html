Fizz

Fizz is a game server browser for Windows 9x/ME/2K/XP. It fully supports Quake, QuakeWorld, Quake 2, Quake 3, Half-Life, Tribes, Tribes 2, Wolfenstein (version 1.x) or only QuakeWorld and Battlefield 1942 (version 2.x). All mods/addons for those games are supported as well as any game based on those engines such as SiN, Kingpin and Soldier of Fortune. There's two versions with the same features except 1.x has built in querying of game servers where as in 2.x it has been moved to DLL files. Run fizz.exe -nosplash to disable to splash screen.

Features

  • Simple and easy to use
  • Small and fast
  • Customisable
  • ICQ integration
  • Filtering of servers
  • Multi-user
  • Many ways to start games
  • No ADs

Old history
Language: Delphi.
Fizz was my first major GUI program and it was created because there was a lack of decent game server browsers at the time. There were very simple requirements, a static server list (usually populated by copying addresses off websites) which could be refreshed, a list of players and joining a game also had to support external programs such as proxies. And naturally, be small, fast and not crash. Back then even Delphi programs were acceptable small, a splash screen to slow down startup was fine and if you're lucky it won't internally combust. Ah well. Fizz is a synonym of effervesce, a word which was mentioned somewhere along the line in Science.
